# say Overview

## Summary

`say` is the baseline greeting skill for this workspace.

## User Value

It gives the user a simple example of how a Developer Dashboard skill can expose:

- a CLI command
- a browser page
- a test suite
- a changelog and skill-local documentation set

## Current Features

- `hello` CLI command prints `hello`
- `/app/say/hello` renders `Hello World`
- Playwright browser test verifies the browser page
- Docker-only verification and 100% coverage are enforced
